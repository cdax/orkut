Novel way of posting code
If you really, really, really want to post code on this community, go to this link:

http://phpfi.com

Enter your code into the big white box, and click on 'save' (lower-right-hand-side corner).

Once the next page is loaded, copy its URL, and paste it in your reply/post/scrap.

e.g., follow this link for the code to the Hello, world. program:

http://phpfi.com/355757
