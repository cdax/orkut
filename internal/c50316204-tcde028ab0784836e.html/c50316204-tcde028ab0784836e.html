While coding in (or googling on) C, you may stumble upon new and interesting stuff about the language that may not have been taught in class. Post all that and more, right under this topic. e.g.,

Clearing the screen

Here's a neat function for clearing the screen before and after each time you run your C program:

system("clear");

Just use this statement as the first and the second-to-last line of main() to run your program on a cleared screen everytime, and to leave the screen clear everytime your program terminates.

e.g.,

int main()
{

system("clear");
..... .....
/*The rest of your code goes here*/
..... .....
system("clear");
return 0;

}

And yes, you don't need to include any header file to use this function.
